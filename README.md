# Jammming

## Overview

Jammming is a web application that allows users to search for songs, create custom playlists, and save them to their Spotify account.

## Features

- **Search Functionality**: Search for songs across the Spotify library
- **Playlist Creation**: Build custom playlists by adding/removing tracks
- **Spotify Integration**: Authenticate and save playlists directly to your Spotify account
- **User-Friendly Interface**: Intuitive design for easy playlist management

## Tech Stack

- React
- Spotify Web API
- JavaScript (ES6+)
- CSS

## Getting Started

### Prerequisites

- Node.js and npm installed
- Spotify Developer account and API credentials

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Add your Spotify API credentials to the configuration
4. Start the development server: `npm start`

## Usage

1. Log in with your Spotify account
2. Search for songs using the search bar
3. Add songs to your playlist
4. Review your playlist and save it to Spotify
